Privacy Policy

DocScanLite • Last updated: 25 Dec 2025

This Privacy Policy explains how DocScanLite (“the app”, “we”, “our”) handles information when you use the app. By using DocScanLite, you agree to the practices described in this policy.

Camera Permission

DocScanLite uses the device camera only to scan documents (e.g., capturing images of pages for PDF/document creation). The camera is accessed only when you choose to use the scan/capture feature.

How Images Are Processed

Captured images are processed on-device to generate scans and PDFs. If you use features that require upload (for example, saving to a cloud service), the app will upload only the files you choose to upload.

Data Selling

DocScanLite does not sell personal data.

Data Sharing

We share data only when necessary to provide the features you request (for example, when you choose to upload files to a cloud storage provider), or if required by law.
